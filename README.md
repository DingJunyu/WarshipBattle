# WarshipBattle  
自主制作  

##操作説明：
-←：左に向かて行く  
-→：右に向かて行く  
-Ⅹ：舵が自動的に元に戻る  
-Ｓ：射撃  
-↑：速度ふやす  
-↓：速度減らす  
-０：テストモードオン／オフ  
-Esc:前画面に戻る/ゲーム終了
